# Pickleball Facility Utilization Dashboard

## Overview
This project analyzes attendance, utilization, and revenue trends across 205 pickleball facility sessions over a four-week period.

## Tools Used
- Tableau Public
- Google Sheets
- GitHub

## Live Dashboard
[View the Tableau Dashboard](https://public.tableau.com/app/profile/tia.elpusan/viz/PickleballFacilityUtilizationDashboard/Dashboard1)

## Key Findings
- Saturday had the highest average fill rate at 98.5%.
- Wednesday had the lowest average fill rate at 58.8%.
- Morning sessions had the highest average fill rate at 84.6%.
- Tia generated the highest revenue among individual coaches in the sample.
